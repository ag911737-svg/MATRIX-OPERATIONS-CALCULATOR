# Matrix Operation Simulator (COA)

A software-based simulator developed in **Scilab** to demonstrate the principles of **Computer Organization and Architecture (COA)** through matrix manipulation. This project illustrates how high-level mathematical operations are translated into memory-mapped data structures and ALU-level computations.

## 📌 Project Overview
This simulator serves as a pedagogical tool to understand:
* **Memory Addressing:** How 2D arrays are mapped to 1D physical memory.
* **Data Representation:** Utilization of the **IEEE 754 Double Precision** format.
* **Instruction Cycles:** The complexity of nested loops and Multiply-Accumulate (MAC) operations in the ALU.

## 🚀 Features
* **Matrix Addition & Subtraction:** Demonstrates linear memory traversal and element-wise processing.
* **Matrix Multiplication:** Highlights computational complexity and intensive register usage.
* **Transpose Operations:** Illustrates the difference between **Row-Major** and **Column-Major** memory access patterns.
* **Dynamic Input:** User-defined dimensions and elements for real-time simulation.

## 💻 Tech Stack
* **Language:** Scilab
* **Environment:** Scilab 2024.0.0 or later
* **Implementation:** Software-based emulator
